    Welcome to fckaweEngine

    This is just a small project to discover java game programming. fckaweEngine aims to be a base library for simple java games. First it should cover things like:

    • configuration handling
    • console logging
    • optimized FPS logic
    • basic bitmap blitting
    • basic custom fonts
    • sprites with some basic physics
    • collision detection (simple boundaries and pixel perfect)
    • keyboard input handling
    • simple sound output
    • tile based levels and maybe a level editor
    • simple menu navigation
    • internationalized user interface (english, german, maybe spanish)

    The idea is to use as few a possible external libraries. To be less dependent and to learn more.
